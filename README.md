# SaaS Sales Dashboard Project

## Project Overview

This project focuses on cleaning, analyzing, and visualizing SaaS sales data using Excel dashboards and pivot tables. The main goal of the project is to identify sales trends, profit performance, customer behavior, and regional business insights.

The project includes:

* Raw sales data
* Cleaned and transformed dataset
* Pivot table analysis
* Interactive dashboard creation

# Important Instructions Before Opening the Project

To access and use this project properly, users must first download the Excel dashboard file to their local system.

## Why Download is Required

The dashboard contains:

* Pivot Tables
* Interactive Charts
* Slicers
* Excel Dashboard Features
* Linked worksheet connections

These features may not work correctly in preview mode, browser mode, or online viewers.

## Steps to Access the Project Properly

1. Download the Excel project file.
2. Open the file using Microsoft Excel.
3. Click "Enable Editing" if prompted.
4. Enable content/macros if required.
5. Navigate to the DASHBOARD sheet to interact with the project.

## Recommended Software

* Microsoft Excel 2016 or later
* Excel 365 (Recommended)

Without downloading the file, some dashboard elements may not load properly or may become inaccessible.
